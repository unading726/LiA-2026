
| **层级**                      | **典型方案（雇主视角）** | **你的可介入行动**                                                                                                  |
| --------------------------- | -------------- | ------------------------------------------------------------------------------------------------------------ |
| **Elimination**             | 淘汰不必要的放射检查     | 与殷教授合作，分析THC的CT/X光使用数据，**识别可被超声替代的非必要检查**，向卫生院的临床医生提出循证建议——这是高影响力的医学物理学贡献                                    |
| **Substitution**            | 用低剂量技术替代       | 调查现有设备的**剂量优化设置**（如CT的自动曝光控制AEC是否开启），很多乡镇技师使用出厂默认参数从未调整——你可以制作一个简单的"剂量优化checklist"                           |
| **Engineering Controls**    | 屏蔽、联锁、通风       | **系统性audit机房物理防护**（使用GBZ 130-2020作为checklist逐项检查），拍摄照片记录缺陷，输出一份带优先级的工程整改建议报告给院方和昆山卫健委——这不需要你施工，但需要你具备识别问题的眼光 |
| **Administrative Controls** | 培训、SOP、警示      | 这是你的核心战场：Zero-Cost Behavioural Safety Toolkit、KAP调查、面对面培训                                                    |
| **PPE**                     | 铅衣、剂量计、护目镜     | Audit PPE存放条件（铅衣是否挂放而非折叠）、检查有效期标签、**设计"剂量计防遗忘"视觉提示系统**（比如机房门口的红绿灯挂牌）                                         |


**Engineering Controls方面你可以做的具体工作**：

**•** 使用GBZ 130-2020作为审计标准，进行**结构化的机房安全walkthrough**

**•** 测量/估算控制台到散射源的距离是否达标

**•** 检查防护门联锁功能是否正常

**•** 这些审计发现写入报告，本身就是高价值的Engineering Controls建议
